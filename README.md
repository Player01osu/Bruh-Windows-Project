# Yuri Web Server

Web server source code for yuri website.

Built using actix framework and yew framework

## Requirements

- Rust 1.58+

## Testing

`cargo build`

`cd ./target/debug/`

`./backend`

Binded on `127.0.0.1:8080`

## Development

Clone this repository

`git clone https://github.com/player01osu/Bruh-Web-Server.git`
